🔒 lockmac

lockmac — macOS 隐私遮罩 / 锁定 / 定时开关，支持 Telegram 远程控制。终端用 `lockmac <命令>`；Telegram 绑定后在对话里用 `/命令`。必须先 `lockmac setup` 设密码（不允许无密码遮罩）。

① 安装

Homebrew（推荐）
brew tap Well365/lockmac && brew install lockmac
一条命令装好,日后 brew upgrade 升级。
.pkg 安装包
# 双击 lockmac-0.5.0.pkg
新机器推荐:已预编译,无需 Xcode。
一键脚本
curl -fsSL https://github.com/Well365/lockMac/raw/main/install.sh | bash
联网一键安装。
pip（开发）
pip install -e lockmac
从源码安装,需 Xcode 命令行工具。

② 给 .pkg/.app 签名 &amp; 常见问题

① 自签名(默认已做,免费)
sudo installer -pkg lockmac-0.5.0.pkg -target /
包内程序已 ad-hoc 签名,任何 Mac(含 Apple 芯片)都能运行。双击「Install lockmac.command」或用此命令安装,均不需付费证书。
② 安装后重新自签名
sudo codesign --force --deep --sign - \ /usr/local/lib/lockmac/* /usr/local/bin/lockmac xattr -dr com.apple.quarantine /Applications/lockmac.app
若二进制被改动/被隔离,出现「已损坏 / 无法验证」无法运行,用这个重签 + 去隔离。
③ 正式签名 + 公证(对外分发零警告)
productsign --sign "Developer ID Installer: NAME (TEAMID)" in.pkg out.pkg xcrun notarytool submit out.pkg --apple-id ID --team-id TEAMID --password PW --wait xcrun stapler staple out.pkg
需 Apple 开发者账号($99/年)。公证要求包内所有可执行文件用 Developer ID Application + hardened runtime 签名。

使用中常见问题

无法验证开发者右键 .pkg / .command → 打开 → 打开;或用 sudo installer;或 xattr -dr com.apple.quarantine <文件> 后再双击。
菜单栏看不到 🔒刘海屏把图标挤到刘海后了——按住 ⌘ 把别的菜单栏图标拖走腾位置,或装免费的 Ice 管理菜单栏。
重启后没升遮罩需 enable_on_boot=on 且装了 com.lockmac 自启 agent。引导默认已开;或跑 lockmac boot on 然后 lockmac start。
录音超时 / 没声音运行监听的 python 没拿到麦克风权限。系统设置→隐私与安全性→麦克风,加入它并打开,再 lockmac tg-restart。
iMessage 收不到命令需「完全磁盘访问」。菜单:设置→完全磁盘访问;把运行监听的 python 加入并打开,再 lockmac tg-restart。

Terminal — zsh
$ lockmac setup

④ 命令参考

遮罩 / 锁定

veil [秒]开隐私遮罩（盖屏防窥，不锁机；可设自动消失秒数）
unveil解除遮罩（需密码；启用 2FA 还需验证码）
lock系统真锁屏（单向，需到机器前输系统密码解）
status查看状态（遮罩 / 密码 / 绑定 / 服务 / 离线锁 / 功能开关）
monitor [秒]监控电脑入侵 开/关(SSH/远程/摄像头/麦克风/USB→TG);`monitor 60` 设检查间隔(默认60s)
eject弹出所有外接 USB 磁盘(diskutil eject)

密码 / 验证

setup首次设置：设密码 + 开机自启（已绑定则一并装监听）
passwd改密码（需输入当前密码）
set-password <密码>直接重设密码（忘记旧密码时用；密码会进命令行历史）
setup-2fa启用 TOTP 两步验证
2fa-off关闭两步验证

定时 / 离线 / 删除

deadman [<签到> <宽限> <动作> [失联]] | off定时开关：无响应/失联自动执行（无参=查看;off=关闭）
offline-lock [<grace> <relock> | off]离线锁：断网持续重锁直到输入密码（无参=查看;off=关闭）
delete add <绝对路径>增:把文件夹加入删除清单（危险路径自动拒绝;别名 purge-add）
delete list | clear | remove <路径>查/清空/移除 删除清单
delete now --yes⚠️ 立即 rm -rf 清单内文件夹(需 --yes);TG 端直接发 /delete 即删

开机自启 / 服务

start一键启动全部服务(遮罩自启 + 监听 + watchdog);已运行则重启
stop停止并卸载全部服务(配置/密码保留)
boot on|off是否开机自动升遮罩
install-agent装遮罩开机自启（LaunchAgent）
uninstall-agent卸载遮罩开机自启
tg-install [--no-watchdog] [--watchdog-interval N]装 Telegram 监听常驻服务（默认含每小时 watchdog）
tg-uninstall卸载监听 + watchdog
tg-restart立即重启监听服务（watchdog 也调它）
menubar [-install|-uninstall]菜单栏图形工具(三语):前台运行 / 装成登录自启
lang [zh|en|ja]设置界面/消息语言(图形菜单栏也跟随;无参=查看)

Telegram / 后端

tg-setup绑定 bot（粘贴 token，自动获取 chat id）
tg-test发一条测试消息
tg-info查看当前绑定(token 打码 + chat id)
tg-unbind解绑 Telegram(清除 token/chat,并停监听)
tg-listen前台跑监听（常驻请用 tg-install）
lock-backend [self|si4locker]切换锁屏后端（无参=查看）
si4lock手动触发 si4locker 锁屏（测试用）
capture [秒]一条命令同时拍照 + 录音(默认 8s;`capture 15` 录 15s;免费、开箱即用)
channel [imessage <h>|ntfy <url>|<n> off|test]告警通道:Telegram + iMessage + ntfy 扇出(无参=查看)
fda检查/引导「完全磁盘访问」(iMessage 收命令需要;弹框打开设置页)
license [<url> <key> | refresh]付费授权:配置/刷新许可(capture 等付费功能需服务器授权;无参=查看)
