📑 文档大纲
① 核心概念
概念是什么
遮罩 (veil)盖住整屏的隐私层,不锁机 —— 远程操作、截图照常,只是别人看不见你屏幕。需密码(+可选 2FA)解除。
系统锁屏 (lock)真锁屏,单向:能远程锁,但只能到机器前输系统密码解。
开机自动升遮罩每次开机/登录自动升遮罩,需输入 lockmac 密码才进入 —— 核心防护(引导默认开启)。
② 安装(双击即可)

拿到 dist/ 文件夹里的 lockmac-0.5.0.pkg 和 Install lockmac.command。

1
双击 Install lockmac.command
第一次被系统拦(“无法验证开发者”)→ 右键 → 打开 → 打开(仅第一次)。
2
输入 Mac 登录密码
弹出密码框,用于安装授权一次。
3
自动完成
菜单栏出现 🔒,设置引导自动弹出。
✅
无需付费签名:包内程序已自签名(ad-hoc),任何 Mac(含 Apple 芯片)都能跑。也可用命令行 sudo installer -pkg lockmac-0.5.0.pkg -target / 安装(无拦截)。
③ 首次设置引导

引导依次弹出,全程不用终端:

步骤说明能否跳过
① 选择语言下拉选 15 种语言(中/英/日/德/法/西/意/葡/俄/韩/阿/泰/瑞典/荷/印地)。必选
② 设置密码解除遮罩用,必填(无「取消」)。不可跳过
③ 开机升遮罩默认开启;不想要点「关闭」。默认开
④ 两步验证 2FA解锁需密码 + 6 位动态码,更安全。下一步
⑤ 绑定 Telegram手机远程锁屏/拍照/告警(见 ⑤)。下一步
⑥ 绑定 iMessage用自己的 Apple ID 收告警/发命令(无需 bot)。下一步
⑦ 自动启动服务引导结尾自动拉起遮罩自启 + 监听 + 看护 + 监控。自动
⑧ 权限提醒提示首次需手动各跑一次以授予系统权限(见 ⑥)。—
✅
设置完毕后,重启电脑登录即自动升遮罩。

⑤ 绑定 Telegram(图形流程)

设置 → 绑定 Telegram(或引导第 ⑤ 步):

1
点「如何获取 token?」
打开图文教程 —— 在 @BotFather 创建 bot 拿到 token。
2
粘贴 token
在框里 ⌘V 粘贴 → 确定。
3
给 bot 发条消息
弹「正在检测绑定…」后,去 Telegram 给你的 bot 发任意一条消息。
4
自动轮询
每 10 秒检测一次,最多 2 分钟;成功自动提示,可「取消」中止。
5
失败处理
超时则留在绑定框,可重输 token 重试或选「下一步」稍后再绑 —— 不会自动跳过。
⑥ 授予系统权限(首次必做)

部分功能需要 macOS 授权,首次从菜单各点一次触发授权弹窗:

功能需要的权限在哪授权
拍照 / 录音摄像头 / 麦克风首次点「拍照+录音」会弹请求
iMessage 收命令完全磁盘访问设置 → 完全磁盘访问(引导打开设置页)
系统锁屏 / 遮罩辅助功能首次可能弹请求
⚠️
授权后再重启验证:登录即自动升遮罩 + 后台服务自启。未授权时相关功能不工作。
⑦ 卸载

双击 dist/ 里的 Uninstall lockmac.command → 输入 yes 确认 → 输入 Mac 密码 → 彻底删除 App、服务、配置。

🗑
卸载会删除配置(密码 / Telegram token)。也可手动:把 /Applications/lockmac.app 拖废纸篓 + 菜单「服务 → 停止全部服务」。
