🤖 获取 Telegram Bot 和 Token

lockmac 用一个你自己的 Telegram 机器人(bot)来远程控制。下面 7 步教你从零拿到 bot 的 token 并完成绑定。

七步拿到 Token

1
打开 Telegram,搜索官方机器人 @BotFather(带蓝色对勾)。
🔍 @BotFather ✔︎
2
给 BotFather 发送命令 /newbot。
/newbot
好的，新机器人。我们给它起个名字吧？请为你的 bot 选择一个名称。
3
输入一个显示名称(任意,如 My Lock Bot)。
My Lock Bot
4
输入用户名,必须以 bot 结尾,且全局唯一(如 mylock_2025_bot)。
mylock_2025_bot
5
BotFather 回复你的 Token。复制它——这是访问密钥,务必保密!
完成！恭喜。使用下面的 token 访问 HTTP API：
8123456789:AAH7k...Qm3vX_example_token
6
打开你新建的 bot(点 BotFather 给的 t.me 链接),给它发任意一条消息(如 hi)。lockmac 需要它来确定你的 chat id。
hi
7
在终端运行 lockmac tg-setup,粘贴 token。它会轮询获取 chat id。
$ lockmac tg-setup
⏳重要:刚用 @BotFather 新建的 bot,通常要等 3–5 分钟才能取到你发的消息。tg-setup 会自动轮询约 3 分钟;若仍没取到,过几分钟再重跑 lockmac tg-setup 即可(消息还在)。

安全 & 常见问题

🔑Token 等于密码:别提交进代码、别发群里。泄露了就在 @BotFather 发 /revoke(或 /token)重置,旧 token 立即失效。
一个 token 能给多人用吗? 一个 token = 一个 bot;一个 bot 本身能和很多人对话(各有不同 chat id)。但 lockmac 只绑定一个 chat(fail-closed,只听你那一个),所以实际是「你一个人控制」。
一台机器能用多个 token/bot 吗? 目前一台机器的 lockmac 只保存一份配置 = 绑一个 bot。要换 bot,重跑 lockmac tg-setup 覆盖即可。同一时刻不支持多个 bot 并存(需要多份配置,属未来增强)。
同一个 token 能被两个程序同时用吗? 不能。Telegram 的 getUpdates 每个 token 同时只允许一个消费者。若 lockmac 和别的程序(如另一个 relay)轮询同一 bot,会互相抢消息并报 409 冲突——给 lockmac 单独建一个 bot。
